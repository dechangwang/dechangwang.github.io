intellij_idea import web

intellij idea导入web项目
  • 当在使用intellij idea 编写web项目时，如果电脑突然宕机，那么此web项目的相关配置就可能会损坏，此时的解决方案为：
    • 右键项目名：remove module
    • 点检file -> new -> module from existing sources->找到项目，下一步后选择import module from external model（maven）接着一直next即可
    • 做完上述步骤后，web项目仍旧不能运行，此时需要在project structure中选择facets 然后添加web模块就可
    • 至于导入项目的具体步骤可参考这里
